In this tutorial, you will learn how to encode different types of rests from the Common Music Notation repertoire using MEI.

Please consult XML basics and minimal MEI structure and/or the Quickstart tutorial if you need more information to complete the task below.

According to the MEI Specification, a rest is "a non-sounding event found in the source being transcribed". Encoding that is simple: Just use a <rest> element, with a @dur attribute.

Dans ce tutoriel, vous apprendrez à encoder différents types de silences du répertoire de la notation musicale classique en utilisant MEI.

Veuillez consulter les bases XML et la structure minimale de MEI et/ou le tutoriel Quickstart si vous avez besoin de plus d'informations pour accomplir la tâche ci-dessous.

Selon la Spécification MEI, un silence est "un événement non sonore trouvé dans la source en cours de transcription". Son encodage est simple : utilisez simplement un élément <rest>, avec un attribut @dur.
